# Руководство пользователя
Программа работает с СУБД **MS SQL Server**.
Для начала работы с программой, нужно запустить СУБД  и само приложение.
Бэкап пазы данных: **rojectDir/materials/backup.bak**
## Аутентификация
После запуска приложения, откроется окно логина.
![Окно аутентификации](https://sun9-44.userapi.com/jVlPd-y2iSuilLqrzPCxfkt-dSva8YdnuQHgDw/PfW8lB1nnQY.jpg)
Здесь есть два поля ввода – Login и Password, куда пользователь приложения должен ввести свои логин и пароль. Также, имеется кнопка log in, нажав на которую, приложение свяжется с базой данных и сверит введённые данные с данными из базы данных. Если данные не совпадают, приложение выведет сообщение об ошибке и предложит пользователю возможность восстановить данные.
При нажатии на кнопку «Забыл пароль», поле ввода Login и поле ввода Password меняют свое назначение на поле ввода номера мобильного телефона и поле ввода номера водительского удостоверения, соответственно
После того, как пользователь введёт свои номер телефона и номер водительского удостоверения, для восстановления логина и пароля ему останется лишь снова нажать на кнопку «Забыл пароль», после чего ему будут выданы его логин и пароль.
![Восстановление пароля]( https://sun9-71.userapi.com/yhuWjzQ-toiwUNSxOPAR4K77nDNi_6D1PPn1HA/05OFczyKz7w.jpg)
После ввода верных данных в поля логин и пароль, приложение переходит в одно из двух состояний: приложение для администратора и приложение для пользователя. Вход в одно из этих состояний зависит от типа профиля, закрепленном за данными логином и паролем.
____
## Пользовательский интерфейс
После ввода логина и пароля, за которыми закреплен тип профиля пользователь, приложение переходит в следующее окно:
![Пользовательский интерфейс]( https://sun9-6.userapi.com/v9GdT_vsJ_aTTGM3rN5ShVw9gZKYCgoLAxqtbQ/sed5L8BbXCY.jpg)
Здесь:
- 1 – Имя пользователя, который произвел вход
2 – Список машин пользователя
3 – Кнопка, с помощью которой осуществляется резервация парковочного места
4,5 – Остаток на счёте клиента
### Резервация парковочного места
Для того, чтобы поставить машину на стоянку, пользователю необходимо выбрать одну из своих машин, которую он хочет поставить на стоянку; нажать на кнопку поставить на стоянку. После нажатия на кнопку, появятся два дополнительных поля: 1) выбор парковки, 2) Поле ввода количества часов, на которые пользователь собирается оставить машину. После ввода количества часов, пользователю будет показана цена стоянки.
![Пользовательский интерфейс]( https://sun9-33.userapi.com/i4Kpu0y4wuW_Tmr_ecS5GPIRF0NF-PfoQdRNQA/ZLHPSvEdwkU.jpg)
Если после ввода пользователем количества часов, цена стоянки превышает остаток на счёте, кнопка «Поставить на стоянку» блокируется и появляется кнопка «Пополнить баланс»
пользователю необходимо ввести номер своей банковской карты и сумму пополнения.
![Пополнение баланса]( https://sun9-23.userapi.com/hjNzVt6I2st-U8p1GIr4uhlJN_bQiYXkPE8jcw/CG-bpnoWSbM.jpg)
Если пользователь выбрал машину, адрес парковки, указал кол-во часов, тогда после нажатия на кнопку «Поставить на стоянку», приложение запросит подтверждение операции и в случае согласия пользователя свяжется с базой данных и зарезервирует парковочное место для пользователя, после чего вся информация о парковочном месте и времени парковки будет выведена в окно приложения.
![Резерв парковочного места](https://sun9-76.userapi.com/JjXRl2lVePw3RyeddWZwCKRXp82Vx2hEKX6H3w/SQCfBi7ZA3E.jpg)
Далее, пользователь может завершить стоянку досрочно и получить денежную компенсацию в размере кол-во неиспользованных часов*2.5,  забрать машину во время, либо, если пользователь забыл забрать машину во время, он может забрать её с опозданием, но на баланс пользователя будет наложен штраф, который высчитывается в зависимости от того, на сколько водитель опоздал.

## Интерфейс админитратора
После ввода логина и пароля, за которыми закреплен тип профиля администратор, приложение переходит в следующее окно:
![Интерфейс администратора](https://sun9-47.userapi.com/JnMtdyh2hEu0PEDriKLJgm9sPsvAqK637MA3eA/lvGE8cIBrYc.jpg)
Здесь:
- 1 – Выбор парковки, по которой будет осуществляться поиск свободных мест
2 – Кнопка, при нажатии на которую администратор выведет список всех зерегистрированных пользователей, после чего администратор может изменить поля «Заблокирован» и «Сумма на счёте». Эти Возможности даны для предоставления администратору прав на блокировку пользователя и решения проблем с непрошедшими транзакциями.

![Интерфейс администратора](https://sun1-97.userapi.com/ZZEzZgwGeVLQ1Qn7mnfN16LbRIJ_Cd3qTeDODA/WMVIcioSqK0.jpg)
- 3 – Кнопка, нажав на которую администратор увидит список свободных мест выбранной парковке. Парковка выбирается в меню 1
4 – Кнопка «Найти пользователя». При нажатии на эту кнопку, открывается дополнительное окно поиска пользователя, которое будет рассмотрено далее.
5 – Кнопка открывает дополнительное окно, в котором администратор производит регистрацию нового пользователя

![Интерфейс администратора](https://sun1-91.userapi.com/yhmlC4Krm67NVTUPleWwFsbQ8oUYqiTFWnDXiA/F-d2FjxHfRc.jpg)
### Поиск пользователя
![Поиск пользователя](https://sun1-85.userapi.com/6FABlsax-5lvio3XPzl8I5taQMVOzQ7PYxLglw/v0YVKxSCTM4.jpg)
Здесь осуществляется поиск человека по одному из параметров
Для примера осуществим поиск пользователя с логином maxim
![Поиск пользователя](https://sun1-29.userapi.com/3KEueL5aKZt3vcAHay74CVaIyPBF17RG9jD8iA/Pd-p2kj6QvI.jpg)
Если поиск удачен и программе удалось найти пользователя, выведется окно с информацией о нём, а так же появятся кнопки добавления новой машины пользователю, удаления машины у пользователя, 
При нажатии на кнопку «Добавить пользователю машину», откроется дополнительное окно в которое администратору необходимо ввести номер и модель машины. если машина с таким номером уже есть в базе, программа выдаст ошибку.
Если водитель захочет удалить свою машину и/или профиль, администратор может нажать соответствующую кнопку.
При удалении автомобиля, нужно нажать на кнопку «Удалить машину», после чего появится выпадающее меню, где администратору необходимо выбрать удаляемое ТС, после чего нужно снова нажать на кнопку «Удалить машину».
При удалении пользователя, достаточно нажать на кнопку «Удалить пользователя».
